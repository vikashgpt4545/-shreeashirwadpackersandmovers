# Project Rules & Guidelines

1. **NO SCRIPTS FOR PAGE OR CONTENT CREATION**:
   - Do NOT use scripts to generate pages or content automatically. Every page and piece of content must be created manually and deliberately.

2. **DON'T ASSUME ANYTHING (CHECK CODE FIRST)**:
   - Always check existing files, code, and folder reality before making any changes or decisions.

3. **LOGO IS ONLY SWASTIK**:
   - The logo image (`assets/images/logo.png`) and site icon (`assets/images/favicon.png`) must ONLY contain the sacred Swastik symbol. Do NOT put company name text inside the logo image itself.

4. **DO NOT OVERUSE COMPANY NAME**:
   - Avoid over-using or stuffing the full company name ("Shree Ashirwad Packers and Movers") across the website content or headers to prevent SEO over-optimization penalties.

5. **TARGET KEYWORD & H1**:
   - Target primary keyword: **"Packers and Movers in Ranchi"**.
   - The main `<h1>` heading on the home page hero section MUST be **"Packers and Movers in Ranchi"**.
